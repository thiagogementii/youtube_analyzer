# YouTube Transcriber & Summarizer 🎥🧠

## 🎯 Objetivo

Script Python para baixar o áudio de um vídeo do YouTube, transcrever com Whisper e gerar um resumo com transformers.

## 📦 Tecnologias

- Python 3.10+
- pytube
- openai-whisper
- transformers (BART)
- Docker & docker-compose
- Makefile

## 🚀 Como usar

### Rodar localmente

```bash
make run
